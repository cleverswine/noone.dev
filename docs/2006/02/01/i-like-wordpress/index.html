I like WordPress

February 1, 2006

The longer I use WordPress, the more I like it. It has quite a few features, but they don’t get in the way of simplicity. I’m finding it easy to add content, pages, and links, all “willy-nilly”. Customizing the appearance isn’t that difficult if you’re familiar with CSS and/or PHP. There are also “plugins” galore, such as these. A good experience so far.
