FriendFeed

June 2, 2008

So I’ve been using FriendFeed for a few months now. It basically aggregates my online activity (mostly via “social” networks like Facebook and Flickr) and then provides a feed for anyone to consume. It’s not a bad idea, but it’s really too bad that we need something like this. I think it’s a sign that the social web is out of control. Anyway, I find it useful to track what my friends are doing on the web, since we have similar interests. To see what it looks like, check my feed here: http://friendfeed.com/cleverswine.

Meanwhile, I twittered for the first time last week. It wasn’t as good as people say it is. I’m unfulfilled.
