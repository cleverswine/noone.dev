Feb 07 2008 In Training II

It’s my fourth day in IBM Websphere training. I’m becoming an expert with Process Server (cough cough). It’s been a long time since I’ve been in a training class that really challenges me. As an old school procedural programmer, it’s a bit mind-blowing to design processes visually. I’m a proponent of simple, straight-forward software design – which Process Server is not. I’m not knocking Process Server, it’s actually quite a cool tool. I can foresee hours of fun with it, once I understand it. I am aging prematurely…
