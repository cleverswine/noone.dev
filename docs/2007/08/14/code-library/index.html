Aug 14 2007 Code Library

I enhanced my code library application by applying an Ext JS front end to it. The coolest part is the tree on the left. You can drag/drop code and categories to rearrange them and the changes are saved asynchronously to the database via ajax. I’m hoping that it’s user-friendly enough that I’ll start using it as a code sample repository.
